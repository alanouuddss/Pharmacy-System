# Pharmacy Management System (Data Structures Project)

A Java-based pharmacy management system built using custom data structures.  
The application manages products, medications, and prescriptions using a Singly Linked List and a Linked-List Queue.

## Features
- Add, remove, and update products
- Manage medications and prescription items
- Track product stock and quantities
- FIFO processing of customer prescription requests (Queue)
- Store products using a custom Singly Linked List
- Display product information and pharmacy operations

## Data Structures Used
- **SinglyLinkedList** — to store and manage products
- **LLQueue** — to manage prescription processing
- **Custom classes:** Product, Medication, Other, Prescription
- **PharmacySystem** — main controller for system operations

## Technologies
- Java (OOP + Data Structures)
- Custom Linked List implementation
- Custom Queue implementation
